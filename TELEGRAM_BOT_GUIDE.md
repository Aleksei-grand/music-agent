# 🤖 Telegram Bot - Руководство

## Настройка

### 1. Создание бота

1. Откройте Telegram и найдите `@BotFather`
2. Отправьте `/newbot`
3. Придумайте имя (например, `MyFlowMusic Bot`) и username (например, `your_music_agent_bot`)
4. Получите **токен** - сохраните его!

### 2. Конфигурация

```bash
# Добавьте в .env
MUSIC_AGENT_TELEGRAM_BOT_TOKEN=your_token_here

# ID администраторов (опционально)
MUSIC_AGENT_TELEGRAM_ADMIN_IDS=123456789,987654321

# Режим: polling (по умолчанию) или webhook
MUSIC_AGENT_BOT_MODE=polling
```

Получить свой ID: найдите бота `@userinfobot`

### 3. Установка зависимостей

```bash
pip install python-telegram-bot>=20.7
```

### 4. Запуск

```bash
# Запуск бота
python run_bot.py

# Или
python -m music_agent.bot
```

## Команды бота

### Основные

| Команда | Описание |
|---------|----------|
| `/start` | Начало работы, приветствие |
| `/sync` | Скачать треки с Suno |
| `/translate` | Перевести тексты (выбор альбома) |
| `/cover` | Сгенерировать обложки |
| `/process` | Обработать аудио |
| `/publish` | Опубликовать (выбор альбома → дистрибьютор) |
| `/status` | Показать статистику |
| `/vault` | История работы |
| `/help` | Справка |
| `/cancel` | Отменить операцию |
| `/about` | О боте |

## Conversation Flow

### Перевод текста
```
1. /translate → Показать список альбомов
2. Пользователь выбирает альбом (callback)
3. Запуск перевода
4. Результат
```

### Публикация
```
1. /publish → Показать альбомы
2. Выбор альбома
3. Выбор дистрибьютора
4. Подтверждение
5. Публикация
```

Бот отправляет уведомления о:
- Завершении sync
- Готовности обложки
- Окончании обработки
- Статусе публикации

## Безопасность

- Проверка `telegram_admin_ids`
- Логирование всех действий в Vault
- Валидация перед операциями
- Подтверждение для важных действий

## Режимы работы

### Polling (по умолчанию)
```bash
# Бот постоянно опрашивает Telegram
# Хорошо для разработки
MUSIC_AGENT_BOT_MODE=polling
```

### Webhook (для продакшена)
```bash
# Telegram присылает обновления на URL
MUSIC_AGENT_BOT_MODE=webhook
MUSIC_AGENT_WEBHOOK_URL=https://your-domain.com
MUSIC_AGENT_WEBHOOK_PORT=8443
```
