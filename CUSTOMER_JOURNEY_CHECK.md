---

## 🔍 Проверка по шагам

### ✅ Шаг 1: Заход через Telegram или Web
**Статус: РАБОТАЕТ**
- Telegram Bot: `/start` → главное меню
- Web UI: `http://localhost:8000` → Dashboard

### ✅ Шаг 2: Скачивание треков с Suno
**Статус: РАБОТАЕТ**
```bash
agent sync
/sync  # в боте
```

**Куда скачивает:**
```
storage/
└── raw/
    └── {suno_track_id}/
        └── audio.mp3          ← исходник с Suno
```

### ✅ Шаг 3: МЕЖДУНАРОДНЫЕ НАЗВАНИЯ + ПРЕВЬЮ
**Статус: ✅ РЕАЛИЗОВАНО**

**Новая система названий:**
```python
# Русское название → Авто-транслит → Можно редактировать → Файл

"Моя Песня" → "Moya Pesnya" → [можно поменять на "My Song"] → "01-My Song (original version).mp3"
"Любовь"    → "Lyubov"      → [или оставить]        → "01-Lyubov (original version).mp3"
"Summer"    → "Summer"      → [уже латиница]        → "01-Summer (original version).mp3"
```

**Поле `intl_title` в базе данных:**
- Автоматически заполняется транслитерацией при создании
- Можно отредактировать в Web UI (`/albums/{id}/edit`)
- Используется при формировании имени файла

**Превью перед обработкой (Telegram):**
```
📋 Предпросмотр обработки

📀 Мой Альбом
👤 Artist Name
🎵 Треков: 3

Будут созданы файлы:
✓ 01-Moya Pesnya (original version).mp3
✓ 02-Lyubov (english version).mp3
✓ 03-Summer (original version).mp3

Настройки:
• Fade-out: 3с
• Нормализация: -14 LUFS
• Формат: MP3

[✅ Обработать] [✏️ Изменить названия] [❌ Отмена]
```

**Редактирование названий (Web UI):**
```
┌──────────────────────────────────────────┐
│ Название (оригинал): Моя Песня           │
│ Международное:      [My Song        ]     │
│                                          │
│ Будет создан файл:                       │
│ "01-My Song (original version).mp3"      │
└──────────────────────────────────────────┘
```

### ✅ Шаг 4: Перекодировка
**Статус: РАБОТАЕТ**
```python
processor.process_track(
    fade_out=3.0,           # ✅ Fade-out 3 сек
    normalize_lufs=True,    # ✅ Нормализация -14 LUFS
    trim_silence=True,      # ✅ Обрезка тишины
    metadata={...}          # ✅ ID3 теги
)
```

**Результат:**
```
storage/albums/{album_id}/
├── 01-My Song (original version).mp3
├── 02-Lyubov (english version).mp3
└── cover.jpg
```

### ✅ Шаг 5: Отрисовка обложек
**Статус: РАБОТАЕТ**
```
/cover → AI Nano-Banana-Pro → 3000x3000
```
