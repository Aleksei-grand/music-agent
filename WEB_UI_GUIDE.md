# 🌐 Web UI - Руководство

## Запуск

```bash
# Базовый запуск
python agent.py web

# Другой порт
python agent.py web --port 3000

# Для разработки (автоперезагрузка)
python agent.py web --reload

# Другой хост
python agent.py web --host 127.0.0.1 --port 8080
```

## Доступ

После запуска откройте в браузере:
```
http://localhost:8080
```

## Страницы

### Dashboard (/) - Главная
- 📊 Статистика (альбомы, песни, обработано, опубликовано)
- ⚡ Быстрые действия (синхронизация, обложки, публикация)
- 💿 Последние альбомы с обложками

### Альбомы (/albums)
- 📀 Список всех альбомов в сетке
- 🖼️ Обложки альбомов
- 📄 Пагинация
- 🔗 Переход к деталям

### Альбом (/albums/{id}) - Детальная
- 🖼️ Большая обложка
- 📋 Список треков
- ℹ️ Метаданные (жанр, статус)
- ⚡ Действия (перевод, обложка, обработка, публикация)

### Обложки (/covers)
- 🎨 Все сгенерированные обложки
- 📋 Альбомы без обложек
- ⚡ Кнопка генерации

### Публикация (/publish)
- 📤 Готовые к публикации альбомы
- ✅ Опубликованные
- 🌐 Выбор дистрибьютора
- 📋 Статус публикаций

### История (/vault)
- 📊 Статистика использования
- 🕐 Последние действия
- 📈 Графики активности

### Настройки (/settings)
- ⚙️ Просмотр текущих настроек
- 📋 Информация о конфигурации

## API Endpoints

```
GET  /api/stats              - Статистика (JSON)
POST /api/sync               - Запуск синхронизации
POST /api/albums/{id}/translate  - Перевод
POST /api/albums/{id}/cover      - Генерация обложки
POST /api/albums/{id}/process    - Обработка аудио
POST /api/albums/{id}/publish    - Публикация
GET  /covers/{id}            - Получить обложку
```

## Технологии

- **Backend**: FastAPI (Python)
- **Frontend**: Alpine.js (реактивность)
- **CSS**: Tailwind CSS (стили)
- **Шаблоны**: Jinja2

## Кастомизация

### Изменить цвета
Редактируйте Tailwind классы в шаблонах:
```html
<!-- Зелёный акцент -->
class="text-green-400"
class="bg-green-600 hover:bg-green-700"
```

### Добавить страницу
1. Создайте шаблон в `web/templates/`
2. Добавьте route в `web/app.py`
3. Добавьте ссылку в `base.html`

### API для фронтенда
Используйте `x-data` и `fetch()`:
```html
<div x-data="{ stats: {} }" x-init="fetch('/api/stats').then(r => r.json()).then(d => stats = d)">
    <span x-text="stats.albums"></span>
</div>
```
