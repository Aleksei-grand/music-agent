# 🎨 Генерация Обложек - Руководство

## Быстрый старт

```bash
# 1. Установить зависимости
pip install Pillow opencv-python

# 2. Проверить статус обложек
python agent.py cover-status

# 3. Сгенерировать для всех альбомов без обложек
python agent.py cover --all

# 4. Или для конкретного альбома
python agent.py cover --album-id 01HQ...
```

## Структура хранения

```
storage/covers/
└── {cover_id}/
    ├── prompt.txt          # Текст промпта
    ├── source.jpg          # Исходное изображение от Poe
    ├── cover_3000.jpg      # Обработанная версия (3000x3000)
    ├── cover_1500.jpg      # Вариант поменьше
    └── metadata.json       # Информация о генерации
```

## Требования дистрибьюторов

Проверка перед публикацией:

```bash
# Проверить конкретный файл
python agent.py cover-validate path/to/cover.jpg
```

**Требования:**
- ✅ Формат: JPEG или PNG
- ✅ Размер: минимум 3000x3000 пикселей
- ✅ Соотношение сторон: 1:1 (квадрат)
- ✅ Цветовой режим: RGB
- ⚠️  **БЕЗ ТЕКСТА** (название будет отображаться платформой)
- ⚠️  **БЕЗ штрих-кодов, логотипов, URLs**

## Примеры команд

```bash
# Сгенерировать всё
python agent.py cover --all

# Только посмотреть что будет сгенерировано
python agent.py cover --all --dry-run

# Перегенерировать существующие
python agent.py cover --all --regenerate

# Другой размер (например, 1500 для превью)
python agent.py cover --all --size 1500

# Проверка файла
python agent.py cover-validate ./my-cover.jpg
```

## Интеграция в workflow

```bash
# Полный цикл:
python agent.py sync           # Скачать треки
python agent.py translate      # Перевести тексты  
python agent.py cover --all    # Сгенерировать обложки
```

## Настройка в .env

```env
# Ключ из https://poe.com/api_key
MUSIC_AGENT_POE_API_KEY=poe-xxx...

# Модели
MUSIC_AGENT_POE_COVER_MODEL=Nano-Banana-Pro
MUSIC_AGENT_POE_TRANSLATION_MODEL=Claude-Opus-4.6
```

## Возможные проблемы

### "Model returned text instead of image"
- Модель не поддерживает генерацию изображений
- Проверьте название модели в .env
- Доступные модели смотрите на poe.com

### "File too large"
- Используйте --size 3000 (это оптимально)
- Качество автоматически настраивается

### Изображение не квадратное
- Автоматически обрезается до центра
- Или растягивается с сохранением пропорций
